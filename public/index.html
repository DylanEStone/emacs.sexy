Emacs is a text editor which you can customize and extend in so many ways. It was written in Lisp, the language from which the Gods wrought the universe.

There's a reason why Emacs was featured in Tron Legacy, it is sexy! (and awesome).

You can use Emacs for all of your text processing needs, internet and social network interactions, hacking, coding, managing to-do's and organizing your daily schedule, play tetris and even provide you with free psychotherapy and counseling with M-x doctor.

GNU Emacs is Free Software, free as in beer and free as in free speech. The original Emacs was written by St. iGNUcius himself - Richard Stallman.

Not only does it give the user the four essential freedoms, its architecture is built in such a way that you have the power to mold it, change it and make it comply to your computering ways.

Installing Emacs

You can install Emacs on almost every operative system out there. The officially supported systems are GNU, GNU/Linux, FreeBSD, NetBSD, OpenBSD, Mac OS X, MS Windows and Solaris.

GNU/Linux

GNU Emacs is available on most GNU/Linux distributions. Use yours system's package manager (apt, yum, pacman, etc.) or download Emacs from the GNU site.

Windows

You can download Emacs for Windows from the GNU servers.

More info on Emacs for Windows on the GNU website.

Mac OS X

Download Emacs for Mac OS X.

Learn Emacs

Mastering Emacs is not the goal, it's the path.
- Me

It's not called the self-documenting real-time display editor for nothing. Emacs includes a great tutorial to teach you how to use Emacs while you're using Emacs. Mind blown? This is just the beginning.

Once you've installed Emacs, start it up. You will be greeted with a welcome screen. From there you can click on the Emacs Tutorial link.

You can also start tasting the tip of the rainbow by pressing Alt + X which will take you to the command mode on the lower screen of Emacs. Then write help-with-tutorial and press Enter. You just entered the first of many Emacs commands. You know that warm feeling in your stomach? Get used to it, because there's more where that came from.
